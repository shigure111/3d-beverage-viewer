About This Application

This application is a web-based 3D beverage container viewer built with Three.js, featuring three different beverage container models: a mug, a cola bottle, and a Sprite bottle. Users can freely select models on the webpage, play animations (including lid opening, bottle squeezing, etc.), adjust lighting and camera views, toggle wireframe and textures, change colors, and enjoy rich 3D interactions.

Purpose

The goal of this application is to demonstrate 3D model loading, rendering, and interaction using WebGL technology (powered by the Three.js framework), helping learners understand how to integrate 3D content into web pages and improve their skills in 3D web development and user interface design.

Main Features

  • Switch between three 3D models (mug, cola bottle, Sprite bottle)
  • Lid opening and closing animations for the mug
  • Squeezing deformation animations for the cola and Sprite bottles
  • Toggle wireframe mode and texture display
  • Lighting control (adjust brightness, toggle on/off, change light angles)
  • Camera view adjustment (reset, top view, front view)
  • Custom model color
  • Material inspection and reset
  • About popup window showing application information

Technologies Used

  • Three.js: 3D scene construction, model loading, animation control, rendering
  • HTML5 / CSS3: Webpage layout and styling
  • JavaScript (ES6): Application logic and interactivity
  • Blender: 3D modeling and export (.glb format)

Project Structure (Site Map)

  • Homepage (model display and control interface)
  • Model selection section
  • Animation control section
  • Lighting adjustment section
  • Camera control section

  • Material debugging section
  • About page
